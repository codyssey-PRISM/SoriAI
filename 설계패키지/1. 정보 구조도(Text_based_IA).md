### A-2-2. 온보딩 메인 프로세스 `/onboarding`

- **구현 상태:** ✅ 완료
- **페이지 구조:**
    - 진행 인디케이터 (상단, 1-5 단계 버튼, 각 버튼 클릭 시 해당 Step으로 이동)
    - Step별 폼 영역
    - 네비게이션 버튼 그룹 (이전/다음)
- **A-2-2-1. Step 1: 보호자 정보**
    - 타이틀 영역
    - 서브타이틀 영역
    - 입력 필드:
        - 이름 (필수)
        - 이메일 (필수)
        - `[인증받기]` 버튼 (이메일 입력 후 활성화)
    - `[다음]` 버튼 (Step 2로 이동, 필수 입력 완료 전까지 비활성화)
- **A-2-2-2. Step 2: 소중한 분 정보**
    - 타이틀 영역
    - 서브타이틀 영역
    - 입력 필드:
        - 이름 (필수)
        - 성별 (필수, 라디오 버튼 그룹)
        - 나이 (필수, 스핀 버튼)
        - 연락처 (필수)
        - 보호자와의 관계 (필수, 드롭다운)
        - 거주형태 (선택, 드롭다운)
        - 주요 질환 (선택, 텍스트 영역, 500자 제한)
    - 네비게이션 버튼 그룹:
        - `[이전]` 버튼 (Step 1로 이동)
        - `[다음]` 버튼 (Step 3로 이동, 필수 입력 완료 전까지 비활성화)
- **A-2-2-3. Step 3: 통화 설정**
    - 타이틀 영역
    - 서브타이틀 영역
    - 입력 필드:
        - 하루 통화 횟수 (선택 버튼 그룹: 1회/2회/3회)
        - 통화 시간 (드롭다운)
        - 시작일 (날짜 선택 버튼)
        - 종료일 (선택, 날짜 선택 버튼, 체크박스로 활성화)
        - 반복 요일 (복수 선택 가능, 요일 버튼 그룹)
    - 네비게이션 버튼 그룹:
        - `[이전]` 버튼 (Step 2로 이동)
        - `[다음]` 버튼 (Step 4로 이동)
- **A-2-2-4. Step 4: 통화 내용**
    - 타이틀 영역
    - 서브타이틀 영역
    - 입력 필드:
        - 기본 질문 선택 (선택형 카드 버튼 그룹, 복수 선택 가능):
            - 식사 여부
            - 복약 여부
            - 기분/컨디션
            - 특별한 일
        - 더 물어볼 내용 (선택, 텍스트 영역, 500자 제한)
    - 네비게이션 버튼 그룹:
        - `[이전]` 버튼 (Step 3로 이동)
        - `[다음]` 버튼 (Step 5로 이동)
- **A-2-2-5. Step 5: 최종 확인**
    - 타이틀 영역
    - 서브타이틀 영역
    - 정보 요약 (Accordion UI):
        - 섹션 1: 보호자님 및 소중한 분 정보 (기본으로 펼쳐진 상태)
        - 섹션 2: 마음 돌봄 건강 정보
        - 섹션 3: 소리 통화 설정
        - 각 섹션에 `[수정하기]` 버튼 (해당 Step으로 이동)
    - 동의 (Consent):
        - 피보호자 본인 동의 (필수, 체크박스)
        - 개인정보 수집 및 이용 동의 (필수, 체크박스)
    - 네비게이션 버튼 그룹:
        - `[이전]` 버튼 (Step 4로 이동)
        - `[동의하고 시작하기]` 버튼 (온보딩 완료, 필수 동의 체크 전까지 비활성화)

### B-4. 통화 상세 리포트 (Full Page) `/call-history/[id]`

- **구현 상태:** ✅ 완료
- **페이지 구조:**
    - 페이지 헤더
    - 메인 콘텐츠 영역 (2컬럼 레이아웃):
        - 좌측 컬럼 (고정 너비 550px, 세로 스크롤)
        - 우측 컬럼 (나머지 공간, 세로 스크롤)
- **B-4-1. 페이지 헤더**
    - `[뒤로가기]` 버튼 (화살표 아이콘, 클릭 시 이전 페이지로 이동)
    - 타이틀: "통화 상세 리포트"
- **B-4-2. 좌측 컬럼 (고정 너비 550px, 세로 스크롤)**
    - **통화 기본 정보 섹션:**
        - 타이틀: "통화 기본 정보"
        - 그리드 레이아웃 (2x2):
            - 통화 일시 (날짜 및 시간)
            - 통화 시간
            - 대상 (어르신 이름, 나이, 관계)
            - 통화 상태 (상태 배지, 색상 구분)
    - **상태 분석 섹션:**
        - 타이틀: "상태 분석"
        - 3개 상태 버튼 (그리드 레이아웃):
            - 좋음 (노란색 배경, 3D 아이콘, 선택 시 강조)
            - 보통 (파란색 배경, 3D 아이콘, 선택 시 강조)
            - 나쁨 (빨간색 배경, 3D 아이콘, 선택 시 강조)
        - 현재 상태만 강조 표시
    - **주요 확인 사항 섹션:**
        - 타이틀: "주요 확인 사항"
        - 체크리스트 형식 (3개 항목):
            - 식사 여부 (초록색 배경, 체크 아이콘)
            - 건강/복약 여부 (초록색 배경, 체크 아이콘)
            - 기분 (초록색 배경, 체크 아이콘)
    - **키워드 태그 섹션:**
        - 타이틀: "키워드 태그"
        - 태그 리스트 (해시태그 형식, 색상 구분)
- **B-4-3. 우측 컬럼 (나머지 공간, 세로 스크롤)**
    - **AI 요약 섹션:**
        - 헤더: 소리 캐릭터 아이콘 + "AI 요약" 타이틀 + "인공지능 분석 리포트" 서브타이틀
        - 배경: 보라색 그라데이션
        - 내용 박스: 흰색 배경, 2-3줄 요약 텍스트 (line-clamp-3)
        - 높이: 통화 기본 정보 섹션과 수직 정렬
    - **대화 전체 로그 섹션:**
        - 헤더: "대화 전체 로그" 타이틀
        - 채팅 UI (내부 스크롤):
            - 배경: 그라데이션 배경
            - 소리 메시지 (좌측 정렬, 소리 캐릭터 프로필, 흰색 말풍선)
            - 어르신 메시지 (우측 정렬, 회색 프로필 아이콘, 보라색 그라데이션 말풍선)
            - 내부 스크롤바, 화면 높이를 넘지 않음
